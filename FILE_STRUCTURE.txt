QUICK START
===========

1. DOWNLOAD
   - Download: test-automation-engine.zip
   - Extract to your desired location

2. INSTALL
   cd test-engine
   npm install

3. RUN TESTS
   npm test
   
   OR run specific test:
   npx ts-node runner.ts examples/starter.test.json

4. VIEW RESULTS
   - Open: artifacts/test-report.html
   - Check: artifacts/test-report.json

QUICK COMMANDS
==============

# Install dependencies
npm install

# Run default test
npm test

# Run specific test file
npx ts-node runner.ts examples/starter.test.json

# Debug mode (see browser)
npx ts-node runner.ts examples/starter.test.json --headless=false

# Slow motion (1 second per action)
npx ts-node runner.ts examples/starter.test.json --slowMo=1000

# Different browser
npx ts-node runner.ts examples/starter.test.json --browser=firefox

# Build TypeScript
npm run build

# Clean artifacts
npm run clean
